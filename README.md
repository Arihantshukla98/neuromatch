# PokéMatch

A premium, interactive Pokémon memory matching game built with React and Vite.

## Features
- **Modern UI**: Smooth animations, glassmorphism, and a high-end indigo dark theme using Tailwind CSS v4.
- **Dynamic Difficulty**: Choose between Easy (4x4), Medium (6x6), and Hard (8x8) grid sizes.
- **Multiplayer & AI**: Play locally against a friend or challenge the AI opponent.
- **Powered by PokéAPI**: Real-time fetching of Pokémon sprites.

## Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4
- **State Management**: React Hooks

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd pokematch
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
